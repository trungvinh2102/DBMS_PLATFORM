# Tech: EVM (Ethereum Virtual Machine)

## Stack

- **Solidty**: v0.8.20+
- **Framework**: Hardhat or Foundry.
- **Client**: Viem + Wagmi.

## Security Checklist

- [ ] Reentrancy Guards present?
- [ ] Access Control (Ownable/AccessControl)?
- [ ] Checked for math overflows (if <0.8.0)?
- [ ] Validated all external calls?
