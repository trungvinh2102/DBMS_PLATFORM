# F&B (Food & Beverage) Domain Reference

## Core Modules

### Menu Management

- Categories, subcategories
- Items (name, description, images, pricing)
- Modifiers and add-ons (extra cheese, size options)
- Combo meals, set menus
- Seasonal/limited items
- Nutritional information, allergens

### Order Management

- Order types: Dine-in, Takeaway, Delivery
- Order lifecycle: received → preparing → ready → served/delivered
- Table management (for dine-in)
- Kitchen display system (KDS)
- Order modifications, special requests

### POS (Point of Sale)

- Cash register integration
- Payment processing (cash, card, e-wallets, QR codes)
- Receipt printing
- Split bills, tips

### Inventory & Kitchen

- Ingredient inventory
- Recipe management (ingredients per item)
- Stock alerts, low stock warnings
- Waste tracking
- Supplier management

### Delivery & Logistics

- Delivery zones, radius
- Delivery fees calculation
- Driver assignment, tracking
- Integration with delivery platforms (GrabFood, ShopeeFood)

## Common Features

### Customer Experience

- Mobile ordering (app, QR code)
- Loyalty programs, stamps, points
- Reservations, waitlist
- Customer feedback, ratings

### Multi-location

- Branch management
- Centralized menu with local variations
- Cross-location reporting
- Staff assignment per location

### Marketing

- Promotions, happy hours
- Discount codes, vouchers
- Push notifications
- Social media integration

## User Roles

- **Customer**: Order, pay, review
- **Cashier**: POS operations
- **Kitchen Staff**: View orders, update status
- **Manager**: Reports, inventory, staff
- **Admin**: Full system access

## Key Metrics

- Revenue per day/week/month
- Average order value (AOV)
- Table turnover rate
- Food cost percentage
- Customer satisfaction (NPS)
- Order fulfillment time
